# Webdim

Aplikasi lengkap CRUD mahasiswa berbasis Express dan Next.js dengan autentikasi JWT.

## Fitur yang tersedia

- Registrasi dan login pengguna
- Proteksi endpoint API menggunakan JWT
- CRUD mahasiswa melalui API dan antarmuka web
- Pencarian mahasiswa berdasarkan nama, NIM, atau jurusan
- Halaman tambah, edit, dan hapus data mahasiswa di frontend Next.js

## Menjalankan aplikasi

1. Jalankan backend:
   - npm install
   - npm start
2. Jalankan frontend:
   - cd frontend
   - npm install
   - npm run dev

Backend berjalan di http://localhost:3000 dan frontend di http://localhost:3001.
